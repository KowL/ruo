**更新时间**: 2026-02-27

---

## 一、项目愿景

**核心定位**: Ruo 是面向个人投资者的 AI 智能投顾副驾，最终形态是一套可输出的 **Agent Skill**，可被交易代理（如陈小群代理）调用。

**技术栈**: FastAPI + React + PostgreSQL + LangGraph + Docker

## 五、SKILL 输出设计草案

### 5.1 目标形态
将 Ruo 平台能力封装为 **Agent 可调用的 Tool/Skill**:

```yaml
# ruo-skill.yaml
name: ruo-trading-assistant
version: 1.0.0
description: AI 智能投顾副驾 - 提供市场情绪、概念分析、个股诊断能力

tools:
  - name: market_sentiment
    description: 获取当前市场情绪指数
    endpoint: /api/v1/market/sentiment
    
  - name: concept_analysis
    description: 概念异动分析与龙头股识别
    endpoint: /api/v1/concept/{id}/analysis
    
  - name: stock_diagnosis
    description: 个股 AI 诊断（行情+基本面+情绪）
    endpoint: /api/v1/analysis/stock/{symbol}
    
  - name: news_intelligence
    description: 新闻情报获取与情感分析
    endpoint: /api/v1/news/intelligence
```

### 5.2 对接方式
- **HTTP API**: Agent 直接调用 Ruo 服务
- **MCP Protocol**: Model Context Protocol 标准接口
- **WebSocket**: 实时推送（预警、异动）
