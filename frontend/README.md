# Frontend 前端应用

> 移动端代码将放在这里

## 技术栈选项

### 方案一：Flutter (推荐)
- 跨平台 iOS/Android
- 性能优异
- Material Design

### 方案二：Uni-app
- Vue 生态
- 多端支持
- 开发效率高

## 目录结构规划

```
/frontend
├── /lib (Flutter) 或 /src (Uni-app)
│   ├── /pages          # 页面
│   ├── /components     # 组件
│   ├── /services       # API 服务
│   └── /models         # 数据模型
└── pubspec.yaml 或 package.json
```

## 核心页面

1. **持仓管理** - 查看和管理股票持仓
2. **新闻情报** - AI 分析的新闻摘要
3. **行情看板** - K线图和技术指标
4. **策略中心** - 交易策略配置
5. **个人中心** - 用户设置
