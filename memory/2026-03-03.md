## Phase 4: 行情数据缓存 + 实时推送系统

### 4.1 行情数据缓存优化

#### 已完成
- `backend/app/models/kline.py`: 行情数据表模型
- `backend/app/services/kline_service.py`: 行情数据服务（查询/保存/均线计算）
- `backend/app/tasks/kline_tasks.py`: Celery定时任务
- `backend/migrate_kline.py`: 数据库迁移脚本

#### 功能
- 优先从数据库读取行情数据
- 智能增量更新，只拉取缺失数据
- 预计算均线（MA5/MA10/MA20/MA60）
- 每晚8点自动更新持仓股票，8:30更新热门股票

### 4.2 实时推送系统

#### 后端
- `backend/app/services/notification_service.py`: 通知服务（飞书Webhook）
- `backend/app/core/websocket_manager.py`: WebSocket连接管理
- `backend/app/api/endpoints/websocket.py`: WebSocket API端点
- `backend/app/tasks/websocket_tasks.py`: WebSocket推送任务

#### 功能
- **WebSocket实时推送**: ws://localhost:8000/ws
- **飞书通知**: 预警触发时发送卡片消息
- **多通道支持**: 价格更新、预警通知、市场概览
- **自动重连**: 前端断线后自动重连

#### 前端
- `frontend/web/src/hooks/useWebSocket.ts`: WebSocket Hook
- `frontend/web/src/components/common/RealtimeNotification.tsx`: 实时通知组件

#### 使用方式
1. 在.env中配置飞书Webhook:
   ```
   FEISHU_WEBHOOK_URL=https://open.feishu.cn/open-apis/bot/v2/hook/xxx
   ```

2. 在需要接收推送的页面添加组件:
   ```tsx
   import RealtimeNotification from '@/components/common/RealtimeNotification';
   
   // 在页面底部添加
   <RealtimeNotification />
   ```
